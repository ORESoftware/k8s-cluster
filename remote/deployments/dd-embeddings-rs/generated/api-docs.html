dd-embeddings-rs

Multi-provider embedding gateway + RAG indexing service. 🔒 routes require a bearer token when one is configured. The anthropic provider id is an alias for voyage (Anthropic ships no embeddings API).

MethodPathSummary
GET/healthzLiveness probe (no dependencies).
GET/readyzReadiness probe; checks Qdrant reachability.
GET/api/docs.jsonThis document, as JSON.
GET/api/docsHuman-readable API docs.
GET/api/providers 🔒List configured embedding providers, default models, and aliases.
POST/api/embeddings 🔒Generate embeddings. Body: { provider, input[], model?, dimensions?, input_type? }.
POST/api/rag/index 🔒Embed documents and upsert into a Qdrant collection.
POST/api/rag/search 🔒Embed a query and retrieve nearest neighbors.
